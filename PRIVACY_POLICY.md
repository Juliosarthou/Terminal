# Política de Privacidad de Terminal App

**Última actualización: 27 de marzo de 2026**

Esta Política de Privacidad describe cómo **Terminal App** (en adelante, "la Aplicación") recopila, utiliza y protege su información personal. Valoramos su privacidad y estamos comprometidos a proteger sus datos personales mientras utiliza nuestra herramienta de SSH y Terminal.

## 1. Recopilación de Información
**Terminal App no recopila, almacena ni transmite ningún dato personal identificable a servidores externos.**
-   **No hay registro de usuario:** No solicitamos nombres, correos electrónicos ni información de contacto para el funcionamiento básico.
-   **Credenciales de SSH:** Todas las contraseñas, claves privadas y configuraciones de conexión se almacenan **únicamente de forma local** en el almacenamiento seguro de su dispositivo.

## 2. Uso de la Información
La información que usted ingresa en la aplicación (como direcciones IP, nombres de usuario y claves) se utiliza exclusivamente para:
-   Establecer conexiones SSH entre su dispositivo y sus servidores.
-   Guardar perfiles de conexión para su conveniencia futura (almacenados localmente).

## 3. Permisos de la Aplicación
Para funcionar correctamente, la aplicación solicita los siguientes permisos:
-   **Internet:** Necesario para realizar las conexiones SSH y comunicarse con los servidores remotos que usted elija.
-   **Almacenamiento:** Necesario para guardar archivos de configuración o claves SSH en el dispositivo (si usted lo solicita).

## 4. Proveedores de Servicios y Terceros
La aplicación utiliza los siguientes servicios de terceros para su correcto funcionamiento:
-   **Shorebird:** Esta aplicación utiliza Shorebird para recibir actualizaciones críticas y parches de seguridad de forma dinámica. Shorebird puede recopilar información técnica no identificable (como la versión de la app y el sistema operativo) para verificar e instalar actualizaciones de código. No recopila datos de sus sesiones de terminal.

## 5. Seguridad de los Datos
Implementamos las mejores prácticas de desarrollo para asegurar que sus datos locales estén protegidos bajo la seguridad nativa del sistema operativo (Android/iOS). Se recomienda encarecidamente que el usuario utilice medidas de seguridad en su dispositivo (como bloqueo de pantalla o biometría) para proteger el acceso a sus llaves SSH.

## 6. Cambios a esta Política
Podemos actualizar nuestra Política de Privacidad periódicamente. Se le notificará de cualquier cambio publicando la nueva política en este repositorio de GitHub o dentro de la aplicación.

## 7. Contacto
Si tiene alguna pregunta sobre esta Política de Privacidad, puede contactarnos a través de este repositorio de GitHub.
